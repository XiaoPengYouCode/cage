# FJW Golden Fixtures

This directory is reserved for small, reviewable FJW regression fixtures.

Current fixture:

- `minimal_mma/`: one deterministic MMA step for a three-variable, one-constraint problem.

Large historical MATLAB/Abaqus runtime outputs are not committed here. When those outputs become available, store only compact fixtures, manifests, and checksums in git; keep full ODB and large `.mat` artifacts outside the repository.
