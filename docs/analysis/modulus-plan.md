# 直杆/螺旋杆单胞等效弹性模量求解方案

## Summary

基于当前仓库的 `Seed 55` 单胞，新增一个“纯 Python 高保真近似有限元”分析链，分别求解直杆单元和螺旋杆单元在 `Z` 向、`Ti-6Al-4V ELI = 115 GPa`、`上下完全粘结压板` 条件下的等效弹性模量。

## Key Changes

### 1. 新增独立分析子系统

新增 `cage/analysis/`，与现有绘图/几何代码解耦，包含：

- `config.py`：分析配置 dataclass，固定本次默认值
- `geometry.py`：把当前直杆/螺旋杆几何转换成“可求解的实体域表示”
- `voxel.py`：把实体域离散成规则体素/规则六面体网格
- `elasticity.py`：3D 小变形线弹性求解，使用 `scipy.sparse` 组装和求解
- `compression.py`：实现上下压板边界条件、反力汇总、等效模量计算
- `report.py`：把结果写成 Markdown + JSON

### 2. 几何定义固定为当前仓库参数

分析对象固定为当前代码里的 `Seed 55` 单胞：

- 立方体边长：`1.0`
- 直杆半径：`0.012`
- 螺旋杆：每根边 `3` 个周期，振幅比 `0.06`，杆径与直杆一致
- 只分析单胞，不分析 `3×3×3` 阵列
- 直杆实体：每条 Voronoi 边建成圆柱体并取并集
- 螺旋杆实体：沿现有 helix centerline 建连续圆管并取并集

### 3. 求解方法固定为纯 Python 体素有限元

实现上采用规则体素六面体网格，而不是外部网格器：

- 在单位立方体内建立规则网格，默认从 `96^3` 起步
- 体素是否属于实体，按“体素中心落在任一杆件实体内”判定
- 每个实心体素视作各向同性 Ti64 ELI 线弹性体
- 组装 3D 线弹性刚度矩阵，使用 `scipy.sparse` 解线性系统
- 对两种结构都做 3 个分辨率级别的收敛检查，默认 `96^3 / 128^3 / 160^3`
- 采用最细两档结果差异 `< 5%` 作为“结果可接受”的收敛标准；若未收敛，报告中明确标注

### 4. 边界条件与输出定义固定

本次模量定义固定为“上下完全粘结压板压缩”：

- 底面接触实体节点：`ux = uy = uz = 0`
- 顶面接触实体节点：`ux = 0, uy = 0, uz = -epsilon`
- `epsilon` 取 `1e-3`，保持在线弹性小应变范围
- 横向自由表面不额外约束
- 反力取顶面所有约束自由度的 `z` 向反力总和
- 等效弹性模量定义：`E_eff = Fz / (A * epsilon)`，其中 `A = 1.0`

### 5. 对外接口与产物

新增 CLI 子命令，避免分析逻辑继续塞进绘图入口：

- `uv run cage modulus --seed 55 --style both`
- 默认只做 `Seed 55`，默认同时输出 `cylinder` 和 `helix`
- 默认结果写到：
  - `docs/analysis/modulus_seed55.md`
  - `docs/analysis/modulus_seed55.json`
- Markdown 报告包含：材料参数、边界条件、网格分辨率、收敛表、两种结构的 `E_z`、差值和比值
- JSON 保存原始数值，便于后续继续做参数扫描

## Assumptions

- 材料默认采用 `3D 打印 Ti-6Al-4V ELI, E = 115 GPa`
- 泊松比默认取 `0.34`
- 本次只做 `Z` 向模量，不输出完整刚度矩阵
- 本次只做 `Seed 55`，不扩展到 `116/49`
- 由于选择了“高保真 + 纯 Python”，这里将其具体落实为“体素化 3D 线弹性有限元”，而不是外部 tetra mesher / CalculiX / Abaqus 路线
